Use IPB SDK inside IPB
Version 1.0 Beta Notes

As of Version 1.0 of IPB SDK, you can use the SDK inside Invision Power Board itself. This allows you to enhance and modify your forum how you want it, with the ease and convinience of IPB SDK.

Installation

The first step is to set up the IPB SDK itself. Take a look at docs/install.html from the downloaded file for more information. Once you have installed the IPB SDK you need to make a small change to index.php to allow you to use the SDK's functions in it.

Open index.php. Find around line 228:

$ibforums->js_base_url = $ibforums->vars['board_url'].'/index.'.$ibforums->vars['php_ext'].'?s='.$ibforums->session_id.'&';

Add After:

// IPB SDK 1.0
require_once ("/path/to/ipbsdk/ipbsdk_class.inc.php");
$SDK = new IPBSDK();

You will have to change /path/to/ipbsdk/ to the path of your IPB SDK installation. Save and Upload. You should now be able to use the SDK's functions in IPB.

Tips

Invision Power Board's code contains alot of classes and functions and you'll probably be using IPB SDK's functions in a function. Remember to put global $SDK; at the start of the function or to refer to functions like $GLOBALS['SDK']->is_loggedin().
