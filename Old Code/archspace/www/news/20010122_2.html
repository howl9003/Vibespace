1/22/01 Contact Protocol
 
We realize that there are bugs still remaining in our game. We ask that you help us during this beta testing period so that we may improve Archspace for everyone. If you encounter what you believe is a bug, please post a message on the bug bulletin board in the Forum section. We ask that you include the following information and use this general format.

Portal Login:
Character Name and Number:
Time of Bug:
Page you were on:
The bug itself: What was the problem you noticed. What message did you
                     receive?
What recent actions had you taken with your character:

If you use this format, it will be easier for us to interpret, and the more information you can provide, the more likely it is that we will be able to locate the problem. Also, be aware that some bugs may seem simple, but they may take time to fix, because there may either be more severe bugs that must be fixed first, or this bug is a symptom of a larger bug, or more complicated than it seems.
