1/23/01 Bugs that you should know about right away.
 
There are two very common problems with the game right now.

The first in the Apache .... Error. This is quite common and is the cause for the second bug. We are uncertain of one of the causes. When there are short down time, we have not pinpointed the reason, but we have clues and are on the trail. When there is a long down time, that means we have found a bug and have had to recompile the game. This second case is good for everyone, because it means the game is at least a little more stable.

The Second bug has been dubbed the Roll-back bug. This occurs often when the first bug occurs. What happens is that recent orders or events are sometimes undone. This can be quite frustrating, we know.

To sum up, it is asked that if you encounter these bugs, please have patience. It is just a matter of time until we find the cause. We are aware of them.
